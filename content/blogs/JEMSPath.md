JMESPath is a query language for JSON

相对于 _jq_，JMESPath 更轻量级。各种编程语言都有对应的封装实现，如 Python 的 _jmespath_ 模块。 或者直接使用 parsel 模块，它对 jemspath 再一次进行了封装。
