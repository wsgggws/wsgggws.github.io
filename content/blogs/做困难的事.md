```
              收益大
                ^               。
                |
                |
                |
                |
                |
        易<--|----------------------->难
                |
                |
              收益小
```

## 为什么

1. 容易且收益大的事情轮不到你；
1. 从长远来看，这是少有人走的路；
1. 如果不去尝试做困难的事情，就只会本能地像个婴儿一样啼哭，连走路都学不会；

## 怎么做

1. 克服舒适圈，当你吃力的时候，你就在上坡!
1. 方法总比困难多，行则将至!
1. 看远点!
