```
              收益大
                ^ .(会是你吗?)      .(try to do here)
                |
                |
                |
                |
                |
        易<--|----------------------->难
                |
                |
              收益小
```

## 为什么

1. 容易且收益大的事情轮不到你；
2. 如果不去尝试做难的事情，你只会像个婴儿一样本能啼哭，走路都学不会；
3. 克服舒适圈！当你感觉吃力的时候，你就在上坡；
4. 方法总比困难多，行则将至!
