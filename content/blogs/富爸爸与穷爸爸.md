罗伯特·清崎的经典之作。它改变的不是投资技巧，而是**思维方式**。

## 核心观念

### 1. 穷人为钱工作，富人让钱为他工作
*   **穷人思维**: 努力学习 -> 找份好工作 -> 赚钱 -> 消费。陷入“老鼠赛跑”的死循环。
*   **富人思维**: 努力学习 -> 建立系统/投资 -> 资产产生现金流 -> 财富增值。

### 2. 资产 vs 负债
这是全书最振聋发聩的定义：
*   **资产 (Assets)**: 能把钱放进你口袋里的东西。（如：房租收入、股息、版税、企业盈利）
*   **负债 (Liabilities)**: 把钱从你口袋里拿走的东西。（如：自住房的房贷、私家车、奢侈品、信用贷）

**富人买入资产，中产阶级买入他们以为是资产的负债。**

### 3. 财富的定义
财富不是指你拥有多少钱，而是指：**如果你今天停止工作，你还能生存多久？**
财富衡量的是时间。当你的**被动收入 > 总支出**时，你就获得了财务自由。

### 4. 重要的四项技能
1.  **财务知识**: 读懂数字，看懂报表。
2.  **投资策略**: 钱生钱的科学。
3.  **市场供给与需求**: 了解市场的运作。
4.  **法律知识**: 了解税收优惠和公司法，利用规则保护财富。

最重要的是：**投资即使不能致富，也要投资那个终身伴随你的资产——你自己。**
