```
              收益大
                ^ .(会是你吗?)      .(try to do here)
                |
                |
                |
                |
                |
        易<--|----------------------->难
                |
                |
              收益小
```

1. 容易且收益大的事情轮不到你；
2. 如果你不去尝试做难的事情，你连走路都学不会（只会像个婴儿一样啼哭吧）；
3. 克服舒适圈！当你感觉吃力的时候，你在上坡，加油；
4. 方法总比困难多!
